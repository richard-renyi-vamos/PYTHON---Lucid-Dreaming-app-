🌙 SHORT SUMMARY:
This is a simple Lucid Dreaming app in Python that helps users build a dreaming routine with reminders, audio cues, and journaling. We'll use tkinter for a GUI, and basic file storage for dream journal entries.

🌀 DEEPER DETAILS:
The app has 4 main features:

🕒 Reality Check Reminders

🎧 Audio Cue Player for Sleep

📔 Dream Journal Entry
